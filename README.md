# Pokemon Finder

Welcome to the Pokemon Finder project! This project allows users to enter the name of a Pokemon and fetch a photo of that Pokemon.

## Technologies Used
- **HTML** - for structure and content
- **CSS** - for styling and layout
- **JavaScript** - for functionality and interactivity

## How to Use
1. Enter the name of a Pokemon in the input field.
2. Click the "enter" button.
3. A picture of the Pokemon will be displayed below.

## Project Structure
- `index.html` - Main HTML file for the application.
- `index.css` - Stylesheet for the application.
- `index.js` - JavaScript file for fetching data and updating the UI.
